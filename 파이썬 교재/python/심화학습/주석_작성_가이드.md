# 주석 작성 가이드

> **"코드는 기계가 실행하기 위해 작성하지만, 주석은 사람이 이해하기 위해 작성합니다."**

## 1. 주석은 왜 중요할까?

1. **미래의 나를 위한 타임머신**
   - 오늘 짠 코드를 일주일 뒤에 보면 기억이 나지 않습니다.
   - 주석은 미래의 내가 코드를 다시 볼 때, 빠르게 기억을 되살려주는 단서가 됩니다.
2. **동료를 위한 배려:**
   - 개발은 혼자 하는 것이 아닙니다. 내 코드를 읽을 동료에게 *"이 코드는 이런 이유로 이렇게 짰어"*라고 말해주는 것이 주석입니다.
3. **'무엇(What)'이 아닌 '왜(Why)':**
   - 코드는 '무엇'을 하는지 보여주지만, 주석은 **'왜'** 그렇게 해야 했는지를 설명합니다.



## 2. 좋은 주석을 작성하는 원칙

### 원칙 1: 코드를 그대로 읽어주지 말기

> 코드를 보고 바로 알 수 있는 내용은 주석으로 달 필요가 없습니다. 
>
> 오히려 코드를 읽는 데 방해가 됩니다.

- **Bad (나쁜 예)**

  ```python
  i = i + 1  # i에 1을 더한다
  ```

- **Good (좋은 예)**

  ```python
  i = i + 1  # 다음 페이지로 이동하기 위해 인덱스 증가
  ```



### 원칙 2: '의도'와 '이유'를 적기

> 이 코드가 존재하는 목적, 혹은 특정 방식을 선택한 이유를 적기

- **Good**

  ```python
  # 사용자 경험을 위해 로딩 시간은 최대 3초로 제한함
  timeout = 3
  ```



## 3. 파이썬 주석 스타일 (Python Style Guide)

파이썬 공식 스타일 가이드(PEP 8)에서는 다음과 같은 규칙을 권장합니다.

1. **한 줄 주석**

   - 코드와 같은 줄에 쓸 때는 코드에서 **최소 두 칸** 띄우고 `#`을 씁니다.
   - `#` 뒤에는 **한 칸**을 띄우고 내용을 씁니다.

   ```python
   x = x + 1  # Border width 값을 보정
   ```

2. **블록 주석**

   - 복잡한 로직을 설명할 때는 코드 윗줄에 작성합니다.
   - 코드와 같은 들여쓰기 레벨을 유지합니다.

   ```python
   # 리스트에서 중복된 도서를 제거하고
   # 가나다순으로 정렬하여 출력합니다.
   unique_books = sorted(list(set(books)))
   ```

### 코드 상세 주석 예시

```python
# -----------------------------------------------------------
# 로직 구현: 보유하지 않은 도서(missing_books) 찾기 및 결과 출력
# -----------------------------------------------------------

# 없는 책을 담을 빈 리스트를 초기화합니다. (결과 저장용 변수)
missing_books = []

# 대여 희망 도서 목록(rental_books)을 하나씩 순회합니다.
for book in rental_books:
    # 현재 확인 중인 도서(book)가 보유 도서 목록(list_of_books)에 없는지 확인합니다.
    if book not in list_of_books:
        # 보유하지 않은 도서라면, 결과 리스트에 추가합니다.
        missing_books.append(book)

# -----------------------------------------------------------
# 결과 출력: 누락된 도서가 있는 경우와 없는 경우를 분기하여 처리
# -----------------------------------------------------------

# 만약 missing_books 리스트에 요소가 하나라도 있다면 (즉, 비어있지 않다면 True)
if missing_books:
    # 누락된 도서 리스트를 순회하며 안내 메시지를 출력합니다.
    for book in missing_books:
        print(f'{book} 을/를 보충하여야 합니다.')
        
# missing_books 리스트가 비어있다면 (즉, 모든 도서가 보유 중이라면 False)
else:
    print('모든 도서가 대여 가능한 상태입니다.')
```
